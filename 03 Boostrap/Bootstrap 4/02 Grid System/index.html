Sistem Grid Boostrap 4

Sistem grid digunakan untuk mengatur jumlah kolom dalam pembuatan layout suatu halaman. Sistem grid pada Bootstrap 4 sama seperti pada Bootstrap 3 yaitu memungkinkan hingga 12 kolom dalam satu halaman.

Class Grid

Sistem grid pada Boostrap 4 mempunyai 5 class, antara lain : .col- digunakan untuk perangkat ekstra kecil - lebar layar kurang dari 576px. .col-sm- digunakan untuk perangkat kecil - lebar layar sama lebih dari atau sama dengan 576px. .col-md- digunakan untuk perangkat medium - lebar layar sama lebih dari atau sama dengan 768px. .col-lg- digunakan untuk perangkat besar- lebar layar sama lebih dari atau sama dengan 992px. .col-xl- digunakan untuk perangkat ekstra besar- lebar layar sama lebih dari atau sama dengan 1200px.

Struktur Dasar Grid

Hampir sama dengan Bootstrap 3 , Pertama yang harus dilakukan adalah buat baris <div class = "row"> , kemudian, tambahkan jumlah kolom yang diinginkan pada class .col - * - * yang sesuai. Perhatikan bahwa angka dalam .col - * - * harus selalu berjumlah 12 untuk setiap baris atau class row. Berikut contoh pembuatan struktur dasar grid.
